## 字段说明

| 字段 | 说明 | 单位 |
|------|------|------|
| ID | 设备唯一标识符 | - |
| 名称 | 设备中文名称 | - |
| 投资成本 | 单位容量一次性投资 | 元/kW 或 元/kWh |
| 运维成本 | 年运维费用 | 元/kW/年 |
| 寿命 | 设备使用年限 | 年 |
| 效率 | 能量转换效率 | - |
| COP | 制冷/制热系数 | - |
| 容量上限 | 最大可安装容量 | kW 或 kWh |
| 碳排放因子 | 单位产出碳排放 | kg CO2/kWh |

## 约束说明

### 容量约束

- 所有设备容量 ≥ 0
- 设备容量 ≤ 容量上限
- 部分设备可能有容量下限（最小经济规模）

### 效率约束

- CHP 设备：发电效率 + 供热效率 ≤ 0.85（典型值）
- 热泵：COP 随外部温度变化（简化模型中取固定值）

### 储能约束

- SOC 范围：0.1 ~ 0.9（保护电池寿命）
- 最大充放电功率：C/2（2小时充满/放空）
