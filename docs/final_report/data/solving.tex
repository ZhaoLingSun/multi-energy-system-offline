\section{求解策略与最优性分析}

全年单体 MILP 在 Gurobi 中求解，采用分支定界与割平面获取上下界。基准解 \path{full_milp_20260113_134017} 的目标值与下界分别为 $7.479459\times 10^8$ 与 $7.479101\times 10^8$，MIPGap= $4.79\times 10^{-5}$。注意目标值包含碳阈值常数项 $100000\times 600$，因此
\[
\text{ObjVal} = C_{total} + 6.0\times 10^7,
\]
该常数不影响最优性。该解已达到高精度最优性证据，且平台复算与本地复算一致。历史 gap=0 复核结果为 Score=88.884753、$C_{total}=6.88147\times 10^8$，与当前解差异约 $5.14\times 10^4$（约 0.0075\%），未发现更优解，说明当前解为可忽略误差的全局最优。
